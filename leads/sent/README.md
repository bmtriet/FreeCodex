# Sent Outreach

This directory stores public-safe logs of autonomous outreach.

Each log should include:

- Date.
- Channel.
- Public URL.
- Why the lead fit.
- Exact message sent.
- Follow-up status.
- Funnel status.

Do not store private contact data, secrets, raw conversations, or sensitive findings.

## Funnel Status Template

- Reply received: no
- Fit-check opened: no
- Qualified lead: unknown
- Audit requested: no
- Paid conversion: no
- Notes: Public-safe summary only. No private contact info, secrets, or raw private messages.

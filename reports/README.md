# Reports

This directory stores public-safe sample reports and sanitized audit artifacts.

Do not store private client reports, secrets, credentials, raw conversations, or confidential findings here unless the client has explicitly approved publication.

## Samples

- `samples/vibe-agent-demo-audit.md` - synthetic demo report showing the kind of launch blockers the audit is designed to catch.
- `samples/freecodex-audit.md` - actual lightweight audit output for this repository.

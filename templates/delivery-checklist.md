# Delivery Checklist

Client:

## Before Audit

- [ ] Repo is public or explicit permission was granted.
- [ ] Scope is clear.
- [ ] Payment received if this is a paid audit.
- [ ] No private credentials were requested.

## During Audit

- [ ] Run local repo audit CLI.
- [ ] Review findings manually.
- [ ] Redact all sensitive evidence.
- [ ] Separate confirmed issues from manual-review hints.

## Delivery

- [ ] Send report within 24-48 hours.
- [ ] Include prioritized next steps.
- [ ] Offer bounded fixes only for high-confidence issues.
- [ ] Ask permission before publishing any sample, testimonial, or anonymized learning.
